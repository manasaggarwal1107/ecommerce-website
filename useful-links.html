1. Return Eligibility

  • You may return items within [30/60/90] days of receiving your order for a full refund or exchange, provided they meet the following conditions:
    • Items must be in their original condition: unused, unworn, and with all original packaging and tags attached.
    • Some items are non-returnable or have specific return requirements, such as [personalized items, clearance items, etc.].

2. How to Initiate a Return

  • To start a return, please follow these steps:
    • Log In to Your Account: Access your order history and select the item(s) you wish to return.
    • Request a Return Authorization: Submit a return request through our online form or contact our customer service team at [support@example.com] or [phone number].
    • Print and Attach the Return Label: Once your return request is approved, we will send you a return label and instructions.
    • Pack and Ship the Item: Carefully package the item, attach the return label, and send it to the address provided.

3. Refund Processing

  • Refunds will be processed within [7-14] business days after we receive and inspect the returned item. The refund will be issued to the original payment method used for the purchase.
  • Please note that shipping charges are non-refundable, and you are responsible for return shipping costs unless the return is due to an error on our part.

4. Exchanges

  • If you wish to exchange an item, please follow the same return process and place a new order for the replacement item.
  • We cannot guarantee the availability of exchanged items, so we recommend placing a new order as soon as possible.
